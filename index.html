This is the base exercise and workouts dataset for myGymBuddy

myGymBuddy is a smart fitness companion app that helps users plan, track, and improve their workouts. It leverages structured data and intelligent recommendations to generate personalized exercise routines based on user goals, experience level, available equipment, and time constraints.

Available dataset endpoints:

/exercises – List of all base exercises with metadata

/workouts – Predefined workout plans composed of exercises
